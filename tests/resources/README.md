# Тестовые ресурсы для Whisper STT

Эта директория предназначена для тестовых аудиофайлов для проверки работы модуля распознавания речи.

## Как использовать

1. Поместите тестовый аудиофайл в эту директорию:
   - Формат: `.wav`, `.mp3` или `.ogg`
   - Рекомендуется: `.wav` с частотой дискретизации 16kHz, моно

2. Запустите тест:
   ```bash
   python tests/test_whisper.py
   ```

## Пример создания тестового файла

Если у вас есть микрофон, вы можете записать тестовый файл:

```bash
# Linux/macOS (требует ffmpeg)
ffmpeg -f alsa -i default -ar 16000 -ac 1 -t 5 test_audio.wav

# Windows (PowerShell)
# Используйте Audacity или другой аудиоредактор
```

## Примечание

Тестовые файлы не включены в репозиторий (добавлены в .gitignore).
Создайте свой тестовый файл для проверки работы модуля.
